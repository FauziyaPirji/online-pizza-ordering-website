# 🍕 Online Pizza Ordering Website

A static multi-page pizza ordering website developed using **HTML** and **CSS** as a first-semester academic project.

## 📋 Overview

This project simulates an online pizza ordering platform with a clean and user-friendly interface. It includes multiple pages to help users explore the menu, view services, and place orders.

## ✨ Features

* Home page
* Menu page
* Services page
* Order page
* Simple and intuitive navigation
* Consistent layout and styling across all pages
* Semantic HTML structure
* Clean user interface design

## 🛠️ Technologies Used

* HTML
* CSS

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/FauziyaPirji/online-pizza-ordering-website.git
```

2. Open the project folder.

3. Launch `home.html` in your browser.
